hibernate จะดีในส่วนของ Programmer แต่ข้อเสียตรง Performance
-hibernate เป็น API 
-entity คลาสเพื่อ Map ตัว db
-สร้างไฟล์ Resource ใว้เก็บ connection 
-การใช้ static ถ้าประกาศมากไปจะกิน Memory เพราะการประกาศ static จะเก็บใว้ที่ Memory(การใช้ static ต้องใส่ final ที่ตัวแปรใว้ด้วย)
-การสร้างคสาสหรือสร้าง bean ควรใส่ serialVersionUID เอาใว้
